# Lista de Compras

## Descrição

Este projeto é uma aplicação web simples para gerenciar uma lista de compras. Permite adicionar itens, marcá-los como concluídos e removê-los da lista. É ideal para aprender conceitos básicos de desenvolvimento web front-end, como manipulação do DOM, eventos em JavaScript e estilização com CSS.

## Tecnologias Utilizadas

- **HTML5**: Estrutura básica da página, formulários e elementos da interface.
- **CSS3**: Estilização visual, incluindo layouts responsivos e uso de fontes externas (Google Fonts: Inter).
- **JavaScript (Vanilla)**: Lógica interativa para adicionar, remover e gerenciar itens da lista, sem frameworks externos.
- **Google Fonts**: Para melhorar a tipografia e aparência da aplicação.

## Como Usar

1. Abra o arquivo `index.html` em qualquer navegador web moderno.
2. Digite o nome do item no campo de entrada e pressione "Adicionar item".
3. Os itens aparecerão em uma lista abaixo, com opções para marcar como concluído (checkbox) ou remover (botão de lixeira).
4. Ao remover um item, uma notificação temporária será exibida.

## Estrutura do Projeto

- `index.html`: Arquivo principal com a estrutura HTML.
- `script.js`: Código JavaScript para a funcionalidade da aplicação.
- `styles/`: Pasta com arquivos CSS:
  - `global.css`: Estilos globais.
  - `index.css`: Estilos específicos da página.
  - `main.css`: Estilos principais.
- `assets/`: Ícones e imagens em SVG.

## Funcionalidades

- Adição de itens com validação (apenas letras, números e espaços).
- Marcação de itens como concluídos.
- Remoção de itens com feedback visual.
- Interface responsiva.

Este projeto demonstra práticas fundamentais de desenvolvimento web e é um ótimo ponto de partida para iniciantes.

# Shopping List

## Description

This project is a simple web application for managing a shopping list. It allows adding items, marking them as completed, and removing them from the list. It's perfect for learning basic front-end web development concepts, such as DOM manipulation, JavaScript events, and CSS styling.

## Technologies Used

- **HTML5**: Basic page structure, forms, and interface elements.
- **CSS3**: Visual styling, including responsive layouts and external fonts (Google Fonts: Inter).
- **JavaScript (Vanilla)**: Interactive logic to add, remove, and manage list items, without external frameworks.
- **Google Fonts**: To enhance typography and application appearance.

## How to Use

1. Open the `index.html` file in any modern web browser.
2. Type the item name in the input field and click "Adicionar item" (Add item).
3. Items will appear in a list below, with options to mark as completed (checkbox) or remove (trash button).
4. When removing an item, a temporary notification will be displayed.

## Project Structure

- `index.html`: Main file with HTML structure.
- `script.js`: JavaScript code for application functionality.
- `styles/`: Folder with CSS files:
  - `global.css`: Global styles.
  - `index.css`: Page-specific styles.
  - `main.css`: Main styles.
- `assets/`: Icons and images in SVG format.

## Features

- Adding items with validation (letters, numbers, and spaces only).
- Marking items as completed.
- Removing items with visual feedback.
- Responsive interface.

This project demonstrates fundamental web development practices and is a great starting point for beginners.
